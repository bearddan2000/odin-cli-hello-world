# odin-cli-hello-world

## Description
This is a POC project to demonstrate odin.

## Tech stack
- odin
- clang
- llvm

## Docker stack
- ubuntu:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
